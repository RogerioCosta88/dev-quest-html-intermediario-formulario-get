Formulários GET

Todas as Tags de input e incluindo as Tags select e button devem estar dentro da Tag form para que o usuário envie suas informações para o backend e/ou frontend.

Existe duas formas de enviar os formulários, uma delas é o get que deve ser especificado no atributo method.

O method definirá como as informações serão enviadas. Usando o get, as informações serão enviadas por parametro na url utilizando query-spring, que é a mesma forma que o Google usa. Todas as informações escolhidas pelo usuário, apareceram na barra de endereço.

Então devemos tomar cuidado ao colocar senha nesse tipo de formulário.
